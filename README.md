# 📊 Tailwind Trades Power BI Analysis

## 🚀 Project Overview
This project presents an **end-to-end business analysis** of the Tailwind Traders dataset using Power BI.  
It focuses on extracting actionable insights related to **sales performance, profitability, product efficiency, and market behavior**.

The dashboard is designed to support **data-driven decision-making** using advanced DAX, data modeling, and interactive visualizations.

---

## 🎯 Objectives
- Normalize multi-currency sales data into USD  
- Analyze profitability using **YTD & QTD metrics**  
- Identify top-performing products and markets  
- Compare **profitability vs order value**  
- Reduce outlier impact using **median-based metrics**  
- Build an **interactive business dashboard**

---

## 🛠 Tools & Technologies
- Power BI  
- DAX (Time Intelligence, Statistical Functions)  
- Power Query  
- Data Modeling  
- Excel  

---

## 📂 Dataset
The dataset includes:
- Countries.xlsx  
- Purchases Dataset.xlsx  
- Tailwind-Traders-Sales.xlsx

## 📈 Key Insights

### 💰 Revenue & Sales
- Total Net Revenue: **~866K USD**
- Median Sales: **~6.3K**
- Sales trends show **fluctuation over time**, indicating inconsistent demand patterns  

---

### 📦 Product Performance
- Top revenue-generating products:
  - Modular Sofa Set (~93K)
  - Motion Sensor Alarm (~72K)
  - Bamboo Plant Pot (~71K)
- A **small number of products contribute significantly** to total revenue  

---

### 🌍 Market Insights
- UAE contributes **~63% of median sales**, dominating other markets  
- UK and USA show strong engagement through **high loyalty points**  
- France and Australia contribute moderately  

---

### 📊 Profitability Analysis
- Average Profit Margin: **~92% (high and stable)**  
- Profit margins are consistent across countries (~91–92%)  
- Occasional sharp drops indicate **potential cost or pricing inefficiencies**  

---

### 📉 Sales Behavior
- Median sales trend shows **peaks followed by decline**  
- Indicates possible **seasonality or inconsistent purchasing patterns**  

---

## 📐 Key Features Implemented
- Currency normalization using exchange-rate modeling  
- Profit, Profit Margin, YTD & QTD calculations  
- Median Sales & Average Order Value (AOV) metrics  
- Profitability vs Order Value scatter analysis  
- Drill-through pages for country-level insights  
- Report-page tooltips for deeper analysis  
- Top-N filtering for focused visualization  

---

## 💡 Business Insights
- Revenue is **concentrated in a few products**, creating dependency risk  
- UAE is the **strongest revenue-driving market**  
- Profit margins are stable but show **unexpected volatility periods**  
- High customer engagement does not always translate into high revenue  
- Median-based analysis provides **more reliable insights than averages**
