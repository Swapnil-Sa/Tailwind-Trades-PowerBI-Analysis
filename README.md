# Tailwind Trades Power BI Analysis

## 📊 Project Overview
This project is an end-to-end **Power BI project** * that's analyzes the **Tailwind Trades dataset** to uncover insights related to revenue performance, profitability, product efficiency, and market behavior.


The focus of this project is not only reporting but **decision-oriented analytics** using advanced DAX and interactive visual design.

---

## 🎯 Objectives
- Normalize multi-currency sales data into USD
- Analyze profitability using YTD and QTD metrics
- Identify high-performing products and markets
- Compare profitability vs order value using scatter analysis
- Reduce outlier bias using median-based analysis
- Build an interactive, executive-ready dashboard

---

## 🛠 Tools & Technologies
- Power BI  
- DAX (Time Intelligence, Statistical Functions)  
- Power Query  
- Data Modeling
- Excel 

---

## 📐 Key Features Implemented
- Currency normalization using exchange-rate modeling  
- Profit, Profit Margin, YTD & QTD analysis  
- Median Sales and Average Order Value (AOV) metrics  
- Scatter plot analysis for profitability vs order value  
- Drill-through pages for country-level product analysis  
- Report-page tooltips for contextual insights  
- Top-N filtering for cleaner visualizations  

---

## 🔍 Key Insights
- Premium markets were identified with high median sales but lower order volumes  
- A small subset of products contributes disproportionately to total revenue  
- Profit margins showed volatility during specific periods, indicating potential pricing or cost inefficiencies  
- Scatter analysis helped differentiate high-revenue markets from high-efficiency markets  
- Median-based metrics provided a more stable view of customer purchasing behavior
